### Clone and Run the App

Clone the app:

```
git clone git@github.com:tokio-k/todo-app-REACT-nest.git
cd todo-app-REACT-nest
```

Install dependencies:

```
yarn
```

Run the app:

```
yarn start
```

http://localhost:4000
